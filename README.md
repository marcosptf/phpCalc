# phpCalc

******************Calculator********************************
*Calculator CLI 0.0.01 by @marcosptf.                      *
*                                                          *
*To Math, do you need run:                                 *
*php index.php [number] + [operator] + [number] + [enter]  *
*                                                          *
*Example:                                                  *
*php index.php 5 + 5                                       *
*                                                          *
*Obs.:                                                     *
*If you want math a multiply, please, put the operator     *
*between simple quotes like example:                       *
*php index.php 5 '*' 5                                     *
*                                                          *
*To run tests:                                             *
*cd tests/                                                 *
*phpunit tests.php                                         *
